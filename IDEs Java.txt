Fases da Execução Java
1. escrever o código fonte (arquivo com a extensão .java)
2. JDK compila o código fonte e gera o arquivo bytecode (arquivo com a extensão .class)
3. a JVM lê o arquivo compilado e as bibliotecas padrão do Java que estão no JRE

Java JDK
é um kit de desenvolvimento Java que inclui tudo o que é necessário para a construção de soluções nessa linguagem de programação

JRE 
ambiente de Execução multiplataforma que fornece as bibliotecas padrões do Java para o JDK compilar seu código e a JVM executá-lo posteriorente

JVM
maquina virtual do Java responsável por executar o bytecode(.class)
